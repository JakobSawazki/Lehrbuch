📌 Namensregeln für Variablen
  • Anfang: Kleinbuchstabe (keine Ziffer). Danach: Buchstaben, Ziffern, Unterstriche.

Schleifen

for – Anzahl steht fest

for zahl in range(1, 11):
    print(zahl * 5)

range(start, ende, schritt) – der Endwert gehört nicht mehr dazu.

while – Bedingung entscheidet

zahl = 1
while zahl < 11:
    print(zahl * 5)
    zahl = zahl + 1

Zähler vorher initialisieren und im Körper verändern – sonst Endlosschleife!

📌 Die goldene Einrückungs-Regel

Nach jedem Doppelpunkt (if, else:, elif, for, while, def) werden alle zugehörigen Anweisungen eine Ebene eingerückt (üblich: 4 Leerzeichen). Die erste nicht eingerückte Zeile gehört nicht mehr zum Block.

Funktionen

Ohne Parameter

def hallo():
    print("Hallo!")

hallo()   # Aufruf

Mit Parametern

def gruessen(p_name):
    print("Hallo,", p_name)

gruessen("Karin")

Mit Rückgabewert

def quadrat(p_zahl):
    return p_zahl * p_zahl

ergebnis = quadrat(7)
print(ergebnis)   # 49

Merkhilfe

  • def = Funktion definieren
  • Parameter = Informationen, die die Funktion empfängt
  • return = Ergebnis an die Aufrufstelle zurückliefern
  • Aufruf = Name + Klammern: quadrat(7)

Sortieren & Suchen (BPE 7)

Bubble Sort

tausch = True
phasen = 0
while tausch:
    tausch = False
    phasen = phasen + 1
    for i in range(0, len(a) - phasen):
        if a[i] > a[i + 1]:
            zs = a[i]
            a[i] = a[i + 1]
            a[i + 1] = zs
            tausch = True

Nachbarn vergleichen und tauschen – Große „steigen auf“. Fertig, wenn eine Phase ohne Tausch bleibt.

Selection Sort

for akt_idx in range(len(a) - 1):
    min_idx = akt_idx
    for j in range(akt_idx + 1, len(a)):
        if a[j] < a[min_idx]:
            min_idx = j
    if akt_idx != min_idx:
        zs = a[akt_idx]
        a[akt_idx] = a[min_idx]
        a[min_idx] = zs

Kleinstes Element des Rests suchen und nach vorn tauschen. Absteigend: > statt <.

Lineare Suche

gefunden = False
for i in range(len(a)):
    if a[i] == gesucht:
        gefunden = True

Jedes Element prüfen – funktioniert auch unsortiert.

Binäre Suche (Array muss sortiert sein!)

gefunden = False
anfang = 0
ende = len(a) - 1
while gefunden == False and anfang <= ende:
    mitte = int((anfang + ende) / 2)
    if a[mitte] == gesucht:
        gefunden = True
    elif a[mitte] > gesucht:
        ende = mitte - 1
    else:
        anfang = mitte + 1

Suchbereich per Mitte-Vergleich halbieren – sehr schnell bei großen Arrays.

Dynamische Datenstrukturen (BPE 7)

StrukturOperationenMerksatz
Verkettete Liste Einfügen, Löschen (Zeiger umhängen) Anker → Knoten (Daten + Zeiger) → … → NULL. Beim Einfügen: erst Zeiger des neuen Knotens setzen, dann den Vorgänger umhängen!
Stapelspeicher (Stack) PUSH() / POP() LIFO – Last in, first out. Nur das oberste Element ist erreichbar. (Backspace, Zurück-Button, Funktionsaufrufe)
Warteschlange (Queue) ENQUEUE() / DEQUEUE() FIFO – First in, first out. Hinten anstellen, vorn entnehmen. (Druckerwarteschlange, Puffer, Pipe)
Baum modellieren (grafisch/tabellarisch) Wurzel oben, Blätter unten; genau ein Weg zwischen zwei Knoten. Binärbaum: max. 2 Kinder, links/rechts unterschieden. Voll: Blatt oder 2 Kinder. Vollständig: voll + alle Blätter auf gleicher Höhe.

Links der Ja-Fall, rechts der Nein-Fall. Leere Zweige (einseitige Auswahl) werden mit ∅ gekennzeichnet.

Regeln

  • Gelesen wird immer von oben nach unten.
  • Jede Anweisung bekommt einen eigenen Block.
  • Leere Blöcke sind nur in Verzweigungen zulässig.
  • In Schleifenkörper und Zweige können beliebige weitere Strukturelemente eingebaut (geschachtelt) werden.

Häufige Fehlermeldungen – und was sie bedeuten

FehlermeldungUrsache und Lösung
SyntaxError: invalid syntax Tippfehler im Code – oft fehlt der Doppelpunkt nach if, else, for, while oder def, oder eine Klammer ist nicht geschlossen.
IndentationError: expected an indented block Nach einem Doppelpunkt fehlt die Einrückung – oder sie ist uneinheitlich (Leerzeichen und Tabs gemischt).
NameError: name 'xyz' is not defined Eine Variable oder Funktion wird benutzt, bevor sie angelegt wurde – häufig ein Tippfehler im Namen (Groß-/Kleinschreibung zählt!).
TypeError: unsupported operand type(s) for /: 'str' and 'int' Es wird mit einer Zeichenkette gerechnet – typisch nach input(). Lösung: mit float() oder int() umwandeln.
ValueError: could not convert string to float Die Eingabe lässt sich nicht in eine Zahl umwandeln – z. B. wurde abc oder 3,14 (mit Komma) eingegeben.
Das Programm „hängt“ und hört nicht auf Endlosschleife: Die while-Bedingung wird nie falsch. Prüfen: Wird die Zählvariable im Schleifenkörper verändert? (Abbruch in Thonny: Stop-Knopf)
