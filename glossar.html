Glossar wichtiger Begriffe

ABDE FGIK LOPR QSTV WZ

A

Algorithmus

Eine eindeutige Schritt-für-Schritt-Anleitung zur Lösung eines Problems.

Beispiel: „Solange kein Baum vor Kara steht, gehe einen Schritt weiter.“

Array

Eine Datenstruktur, die mehrere Werte in einer festen Reihenfolge speichert. In Python wird dafür meistens eine Liste verwendet.

noten = [2, 1, 3]

Alternative

Eine Entscheidung im Programm. Je nach Bedingung wird ein anderer Block ausgeführt.

Python: if, elif, else

B

Baum

Eine hierarchische Datenstruktur aus Knoten. Oben steht die Wurzel, darunter folgen Kindknoten und Blätter.

Bedingung

Eine Aussage, die in Python entweder True oder False ergibt.

Beispiel: alter >= 18

Binäre Suche

Suchverfahren für sortierte Arrays. In jedem Schritt wird nur noch die linke oder rechte Hälfte weiter betrachtet.

Binärbaum

Baum, bei dem jeder Knoten höchstens zwei Kindknoten besitzt: ein linkes und ein rechtes Kind.

Boolean

Datentyp für Wahrheitswerte. Es gibt nur True und False.

Bubble Sort

Sortierverfahren, bei dem benachbarte Werte verglichen und bei falscher Reihenfolge getauscht werden.

D

Datentyp

Die Art eines Werts, zum Beispiel Text, Ganzzahl, Dezimalzahl oder Wahrheitswert.

Python: str, int, float, bool

Datenstruktur

Eine Art, mehrere Daten so zu speichern, dass ein Programm gut damit arbeiten kann.

Beispiele: Array, verkettete Liste, Stack, Queue, Baum

Debugger

Werkzeug, mit dem ein Programm Schritt für Schritt ausgeführt und beobachtet wird.

E

Eingabe

Daten, die eine Nutzerin oder ein Nutzer in ein Programm eingibt.

Python: input("Name: ")

Einrückung

Leerzeichen am Zeilenanfang. In Python zeigt die Einrückung, welche Zeilen zu einem Block gehören.

EVA-Prinzip

Grundmuster vieler Programme: Eingabe → Verarbeitung → Ausgabe.

F

float

Datentyp für Dezimalzahlen. In Python wird ein Punkt statt eines Kommas verwendet.

preis = 2.99

FIFO

„First in, first out“: Das zuerst eingefügte Element wird zuerst wieder entnommen. Das ist das Prinzip einer Queue.

Funktion

Ein benannter Programmbaustein, der bei Bedarf aufgerufen werden kann.

def quadrat(p_zahl):

G

GUI

Grafische Benutzeroberfläche mit Fenstern, Textfeldern, Buttons und Labels.

Ganzzahl

Zahl ohne Nachkommastellen. Der Python-Datentyp heißt int.

I

Initialisierung

Eine Variable bekommt zum ersten Mal einen Wert.

summe = 0

Index

Die Nummer eines Speicherplatzes in einem Array. In Python beginnt der erste Index bei 0.

namen[0] ist das erste Element.

Iteration

Ein einzelner Durchlauf einer Schleife.

K

Kommentar

Text im Programm, der von Python ignoriert wird. Kommentare erklären den Code.

# Das ist ein Kommentar

Knoten

Ein einzelnes Element in einer verketteten Liste oder in einem Baum. Ein Knoten enthält Daten und oft Verweise auf andere Knoten.

Konsole

Textbereich, in dem ein Programm Eingaben annimmt und Ausgaben anzeigt.

L

Liste

Eine geordnete Sammlung von mehreren Werten.

zahlen = [2, 4, 6]

LIFO

„Last in, first out“: Das zuletzt eingefügte Element wird zuerst wieder entnommen. Das ist das Prinzip eines Stacks.

Lineare Suche

Suchverfahren, bei dem die Elemente der Reihe nach geprüft werden, bis der gesuchte Wert gefunden wurde oder das Array endet.

Logischer Operator

Verknüpft oder verändert Bedingungen: and, or, not.

O

Operator

Ein Zeichen oder Wort, das mit Werten arbeitet.

+, *, ==, and

P

Parameter

Eine Information, die beim Aufruf an eine Funktion übergeben wird.

def gruessen(p_name):

Programm

Eine Folge von Anweisungen, die der Computer ausführt.

R

Rückgabewert

Das Ergebnis, das eine Funktion mit return an die Aufrufstelle zurückgibt.

Q

Queue

Warteschlange: Neue Elemente werden hinten angefügt, vorne wird entnommen. Die Queue arbeitet nach dem FIFO-Prinzip.

S

Schleife

Eine Kontrollstruktur, die Anweisungen wiederholt.

Python: for oder while

Schreibtischtest

Man führt ein Programm gedanklich aus und notiert die Werte der Variablen Schritt für Schritt.

Selection Sort

Sortierverfahren, bei dem immer das kleinste Element des unsortierten Rests gesucht und nach vorne getauscht wird.

Sequenz

Anweisungen werden nacheinander von oben nach unten ausgeführt.

Stack

Stapelspeicher: Neue Elemente werden oben abgelegt und auch oben wieder entnommen. Der Stack arbeitet nach dem LIFO-Prinzip.

Struktogramm

Grafische Darstellung eines Programmablaufs. Es zeigt Sequenzen, Schleifen und Alternativen.

T

Traceback

Fehlerbericht von Python. Die letzte Zeile nennt meist den Fehlertyp.

V

Variable

Ein Name für einen gespeicherten Wert. Man kann sich eine Variable wie eine beschriftete Schublade vorstellen.

alter = 17

Vergleichsoperator

Vergleicht zwei Werte und liefert True oder False.

<, >=, ==, !=

Verkettete Liste

Datenstruktur aus Knoten, bei der jeder Knoten auf den nächsten Knoten verweist. Dadurch kann man Elemente flexibel einfügen und entfernen.

W

Wahrheitswert

Ein Wert, der nur wahr oder falsch sein kann: True oder False.

Wertzuweisung

Eine Variable bekommt mit = einen Wert.

punkte = 12

Wurzel

Der oberste Knoten eines Baums. Von der Wurzel aus erreicht man alle anderen Knoten.

Z

Zeiger

Ein Verweis auf ein anderes Element, zum Beispiel vom aktuellen Knoten einer verketteten Liste zum nächsten Knoten.

Python-Befehle und Schlüsselwörter

Ein- und Ausgabe

print()

Gibt Text, Zahlen oder Variablen in der Konsole aus.

name = "Kara"
print("Hallo", name)

input()

Liest eine Tastatureingabe ein. Wichtig: Das Ergebnis ist immer ein Text.

name = input("Name: ")
print("Willkommen", name)

Umwandeln und Prüfen

int()

Wandelt einen passenden Wert in eine Ganzzahl um.

anzahl = int(input("Anzahl: "))

float()

Wandelt einen passenden Wert in eine Dezimalzahl um.

preis = float(input("Preis: "))

str()

Wandelt einen Wert in Text um, damit er mit anderen Texten verbunden werden kann.

alter_text = str(17)

type()

Zeigt den Datentyp eines Werts oder einer Variable an.

zahl = 3.5
print(type(zahl))

round()

Rundet eine Zahl auf eine bestimmte Anzahl von Nachkommastellen.

preis = round(3.14159, 2)

len()

Zählt Elemente, zum Beispiel Zeichen in einem Text oder Werte in einer Liste.

wort = "Python"
print(len(wort))

Kontrollstrukturen

if

Führt einen Block nur aus, wenn die Bedingung wahr ist.

if alter >= 18:
    print("volljährig")

else

Fängt den Nein-Fall einer Verzweigung auf.

if alter >= 18:
    print("ja")
else:
    print("nein")

elif

Prüft eine weitere Bedingung, wenn die vorherige Bedingung falsch war.

if note == 1:
    print("sehr gut")
elif note == 2:
    print("gut")

for

Wiederholt einen Block für alle Werte einer Folge.

for zahl in range(1, 6):
    print(zahl)

while

Wiederholt einen Block, solange die Bedingung wahr ist.

zahl = 1
while zahl <= 5:
    print(zahl)
    zahl = zahl + 1

range()

Erzeugt eine Zahlenfolge für eine Schleife. Der Endwert gehört nicht mehr dazu.

for i in range(0, 10, 2):
    print(i)

Funktionen und Module

def

Definiert eine eigene Funktion.

def begruessen(p_name):
    print("Hallo", p_name)

return

Gibt ein Ergebnis aus einer Funktion zurück.

def quadrat(p_zahl):
    return p_zahl * p_zahl

import

Lädt ein Modul, damit zusätzliche Funktionen verwendet werden können.

import random
zahl = random.randint(1, 6)

from ... import ...

Lädt bestimmte Bausteine direkt aus einem Modul.

from random import randint
zahl = randint(1, 6)

Listen und Zufall

list()

Wandelt eine Folge in eine Liste um, zum Beispiel zum Anzeigen von range().

zahlen = list(range(1, 6))
print(zahlen)

randint()

Erzeugt eine zufällige Ganzzahl im angegebenen Bereich, inklusive Start und Ende.

from random import randint
wuerfel = randint(1, 6)

Arrays, Suchen und Sortieren (BPE 7)

Index-Merker

Die erste Zelle hat den Index 0. Der letzte gültige Index ist immer len(array) - 1.

120 71 192 43

[]

Legt ein leeres Array an, das später gefüllt werden kann.

zahlen = []

[wert1, wert2]

Legt ein Array direkt mit Startwerten an.

namen = ["Mia", "Leo", "Ali"]

append()

Fügt ein Element am Ende des Arrays an.

zahlen.append(42)

array[index]

Liest oder verändert ein bestimmtes Element. Achtung: Der erste Index ist 0.

print(namen[0])
namen[1] = "Lina"

insert()

Fügt ein Element an einer bestimmten Position ein. Die folgenden Elemente rücken nach rechts.

farben.insert(1, "gruen")

pop()

Entfernt das Element an einem Index und gibt es zurück.

entfernt = farben.pop(2)

remove()

Entfernt den ersten passenden Wert aus einem Array.

farben.remove("rot")

len(array)

Liefert die Anzahl der Elemente. Besonders wichtig für Schleifen über Arrays.

for i in range(len(zahlen)):
    print(i, zahlen[i])

Werte tauschen

Beim Sortieren werden zwei Werte häufig mit einem Zwischenspeicher getauscht.

temp = a[i]
a[i] = a[j]
a[j] = temp

Lineare Suche

Prüft die Werte von links nach rechts. Funktioniert auch bei unsortierten Arrays.

position = -1
for i in range(len(a)):
    if a[i] == gesucht:
        position = i

Binäre Suche

Halbiert den Suchbereich. Voraussetzung: Das Array ist sortiert.

m = int((start + ende) / 2)
if gesucht < a[m]:
    ende = m - 1
else:
    start = m + 1

Stack / Queue

Mit Python-Listen lassen sich Stapel und Warteschlangen modellieren.

stack.append("A")   # push
oben = stack.pop()  # pop

queue.append("A")   # enqueue
vorn = queue.pop(0) # dequeue

Datenbankbegriffe und SQL (BPE 6)

Datenbank

Strukturierte Sammlung zusammengehöriger Daten, die dauerhaft gespeichert und gezielt ausgewertet werden kann.

Datenbankmanagementsystem (DBMS)

Software zum Erstellen, Speichern, Prüfen und Abfragen von Datenbanken, zum Beispiel MySQL.

Entität und Entitätstyp

Eine Entität ist ein konkretes Objekt wie das Buch mit der ID 17. Der Entitätstyp beschreibt die gemeinsame Art, hier BUCH.

Attribut

Eigenschaft eines Entitätstyps, zum Beispiel titel oder erscheinungsjahr bei einem Buch.

Beziehung und Kardinalität

Eine Beziehung verbindet Entitätstypen. Die Kardinalität beschreibt mit 1 und n, wie viele Objekte beteiligt sein können.

Relation

Tabelle im relationalen Modell. Eine Zeile entspricht einem Datensatz, eine Spalte einem Attribut.

Primärschlüssel

Attribut oder Attributkombination, die jeden Datensatz eindeutig identifiziert.

person_id

Fremdschlüssel

Attribut, das auf den Primärschlüssel einer anderen Relation verweist und so Tabellen verbindet.

Redundanz

Unnötige Mehrfachspeicherung derselben Information. Sie verbraucht Platz und kann zu widersprüchlichen Daten führen.

Dritte Normalform (3NF)

Ziel einer sinnvollen Tabellenstruktur: Jedes Nichtschlüsselattribut hängt vom Schlüssel, vom ganzen Schlüssel und von nichts anderem ab.

Datenkonsistenz

Daten sind vollständig, widerspruchsfrei und entsprechen den vereinbarten Regeln.

Referentielle Integrität

Ein Fremdschlüssel darf nur auf einen vorhandenen Datensatz verweisen oder – falls erlaubt – leer sein.

SQL

Structured Query Language: Sprache zum Definieren, Verändern und Abfragen relationaler Datenbanken.

JOIN

Verknüpft Datensätze aus mehreren Tabellen über zusammengehörige Schlüssel.

Wichtige SQL-Bausteine

CREATE TABLE

Legt eine Tabelle samt Spalten und Regeln an.

CREATE TABLE fach (
    fach_id INT PRIMARY KEY,
    name VARCHAR(50) NOT NULL
);

INSERT INTO

Fügt einen Datensatz ein.

INSERT INTO fach (fach_id, name)
VALUES (1, 'Informatik');

SELECT ... WHERE

Wählt Daten aus und grenzt Treffer ein.

SELECT name
FROM fach
WHERE fach_id = 1;

JOIN ... ON

Verbindet mehrere Tabellen über passende Schlüssel.

SELECT person.name, ausleihe.buch_id
FROM ausleihe
JOIN person
  ON ausleihe.person_id = person.person_id;

Graphen und gesellschaftliche Begriffe (BPE 8)

Algorithmische Vorauswahl

Ein Algorithmus entscheidet anhand von Daten und Regeln, welche Informationen zuerst oder überhaupt angezeigt werden.

Filterblase

Ein eingeschränkter Informationsraum, der entstehen kann, wenn Empfehlungssysteme überwiegend Inhalte zeigen, die zu bisherigen Interessen passen.

Graph

Ein Modell aus Knoten und Kanten. Damit lassen sich zum Beispiel Freundschaften, Verkehrsverbindungen oder Abhängigkeiten darstellen.

Knoten

Ein Objekt in einem Graphen, etwa eine Person, ein Bahnhof oder eine Internetseite.

Kante

Eine Verbindung zwischen zwei Knoten. Sie kann gerichtet und mit einem Wert gewichtet sein.

Grad eines Knotens

Anzahl der Kanten, die an einem Knoten anliegen. Bei gerichteten Graphen unterscheidet man Eingangs- und Ausgangsgrad.

Gerichteter Graph

Graph, dessen Kanten eine Richtung besitzen. Ein Folgen auf einer Plattform muss zum Beispiel nicht gegenseitig sein.

Gewichteter Graph

Graph, dessen Kanten Werte tragen, etwa Entfernungen, Fahrzeiten oder Kosten.

Weg und Kreis

Ein Weg verbindet eine Folge von Knoten über Kanten. Endet ein Weg wieder am Startknoten, bildet er einen Kreis.

Transparenz

Nachvollziehbarkeit der Daten, Regeln und Gründe, die zu einer algorithmischen Auswahl oder Entscheidung führen.

Fairness und Wertneutralität

Algorithmen sind nicht automatisch neutral: Daten, Ziele und Gewichtungen können einzelne Gruppen bevorzugen oder benachteiligen.

Kontrollmöglichkeit

Möglichkeit, eine algorithmische Entscheidung zu prüfen, zu korrigieren oder durch einen Menschen überprüfen zu lassen.
