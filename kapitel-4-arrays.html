4.1Warum Datenstrukturen?

Stellen Sie sich einen Kleiderschrank vor, in dem alle T-Shirts, Bücher und Hefte wild durcheinanderliegen – jede Suche dauert ewig. Wer dagegen T-Shirts nach Farbe und Bücher nach Fächern ordnet, findet alles mit einem Handgriff. In der Informatik ist das genauso: Ohne Ordnung geht nichts.

📌 Merke: Datenstruktur

Eine Datenstruktur ist ein Objekt zur Speicherung und Organisation von Daten. Die Daten werden so angeordnet und verknüpft, dass der Zugriff auf sie und ihre Verwaltung effizient möglich sind – mit kurzem Programmcode, wenig Rechenzeit und geringem Speicherbedarf.

Die fünf wichtigsten Datenstrukturen im Überblick – welche man wählt, hängt immer vom Zweck ab:

DatenstrukturKern-MerkmalTypisches Beispiel
Array Elemente gleichen Typs direkt hintereinander im Speicher; Zugriff über fortlaufende Index-Werte (0, 1, 2, …) – direkter Zugriff auf jedes Element. Tägliche Temperaturmessungen im Krankenhaus
Verkettete Liste Elemente (Knoten) verweisen mit Zeigern aufeinander; Einfügen und Entfernen jederzeit möglich, Zugriff nur schrittweise. Haltungsnoten beim Skisprung-Wettkampf (beste/schlechteste wird gestrichen)
Stapelspeicher (Stack) Elemente werden nur oben aufgelegt und oben entnommen – LIFO (Last in, first out). Backspace-Taste löscht das zuletzt getippte Zeichen
Warteschlange (Queue) Elemente werden hinten angefügt und vorn entnommen – FIFO (First in, first out). Wartenummern in der Arztpraxis
Baum Bildet hierarchische Strukturen ab: Knoten verweisen auf untergeordnete Knoten (Eltern → Kinder), oben steht die Wurzel. Organigramm eines Unternehmens

4.2Das Array – ein Schrank mit Schubladen

Ein Array speichert nicht nur einen Wert, sondern eine ganze Reihe von Werten. Das Bild dazu kennen Sie schon aus Kapitel 2 – nur größer: Ein Array ist wie ein Schrank mit durchnummerierten Schubladen. In jeder Schublade liegt ein Wert, und über die Nummer kommt man jederzeit direkt heran.

namensliste =
"Uli"
0
"Anna"
1
"Emil"
2
"Sophia"
3
"Luise"
4
"Hans"
5
📌 Merke: Der Index
  • Die Nummerierung der „Schubladen“ heißt Index – die Speicherplätze sind indiziert.
  • Die Zählung beginnt immer bei 0: Die erste Schublade hat den Index 0, die sechste den Index 5.
  • Das n-te Element hat also den Index n − 1.
ℹ️ Statisch, dynamisch – und was macht Python?

Bei statischen Arrays muss die Anzahl der Elemente von Anfang an festgelegt werden und kann nie mehr geändert werden. Dynamische Arrays können dagegen wachsen und schrumpfen. In Python werden Arrays mit sogenannten Listen umgesetzt – die sind dynamisch. Der Einfachheit halber sprechen wir weiterhin von Arrays.

4.3Arrays anlegen (Deklaration & Initialisierung)

Möglichkeit 1: Das Array wird erzeugt und direkt mit Werten gefüllt:

# Deklaration und Initialisierung
namensliste = ["Uli", "Anna", "Emil", "Sophia"]

Möglichkeit 2: Zuerst wird ein leeres Array erzeugt, dann werden die Werte mit der Funktion append() nach und nach am Ende angehängt: 

# Deklaration und Initialisierung
himmelsrichtungen = []

# Zuweisung
himmelsrichtungen.append("Nord")
himmelsrichtungen.append("Ost")
himmelsrichtungen.append("Süd")
himmelsrichtungen.append("West")

Im Ergebnis ist das Array ["Nord", "Ost", "Süd", "West"] entstanden.

💡 Wann welche Möglichkeit?

Möglichkeit 1 passt, wenn die Werte von Anfang an feststehen. Möglichkeit 2 brauchen Sie, wenn die Werte erst nach und nach entstehen – z. B. bei Benutzereingaben: zahlen.append(int(input("Zahl: ")))

4.4Zugriff mit dem Index

Auf ein einzelnes Element greift man mit dem Array-Namen und dem Index in eckigen Klammern zu: 

himmelsrichtungen = ["Nord", "Ost", "Süd", "West"]
print(himmelsrichtungen[3])   # Ausgabe: West
print(himmelsrichtungen[0])   # Ausgabe: Nord
West
Nord

Elemente verändern

Die Elemente sind nicht fest – über den Index können bestehende Werte überschrieben werden. Verfolgen Sie, wie sich das Array training Zeile für Zeile verändert:

Python-CodeInhalt des Arrays danach
training = ["Montag", "Donnerstag"]["Montag", "Donnerstag"]
training[0] = "Dienstag"["Dienstag", "Donnerstag"]
training.append("Freitag")["Dienstag", "Donnerstag", "Freitag"]
training[2] = "Samstag"["Dienstag", "Donnerstag", "Samstag"]
⚠️ Achtung: IndexError

Der Zugriff auf einen Index, den es nicht gibt (z. B. training[5] bei nur 3 Elementen), führt zur Fehlermeldung IndexError: list index out of range. Häufigste Ursache: vergessen, dass die Zählung bei 0 beginnt!

Mit welchem Ausdruck greifen Sie auf das vierte Element des Arrays werte zu?

4.5Die Länge eines Arrays: len()

Oft muss ein Programm wissen, wie viele Elemente ein Array enthält. Das ermittelt die Funktion len() (von englisch length = Länge): 

training = ["Montag", "Mittwoch", "Freitag"]
anzahl = len(training)
print("Trainingstage:", anzahl)
Trainingstage: 3
📌 Merke

len(array) liefert die Anzahl der Elemente. Das letzte Element hat dann den Index len(array) - 1.

4.6Arrays mit der for-Schleife durchlaufen

Betrachten Sie dieses Array:

zahlenliste = [12, 9, 17, 31, 8, 24, 51, 67, 2, 34]

Um alle Werte auszugeben, könnte man zehnmal print(zahlenliste[0]), print(zahlenliste[1]), … schreiben. Bei 10 000 Zahlen wären das 10 000 Zeilen – das muss einfacher gehen! Hier hilft die bekannte for-Schleife:

Python-Programm

himmelsrichtungen = ["Nord", "Ost", "Süd", "West"]

for i in range(len(himmelsrichtungen)):
    print(himmelsrichtungen[i])

Struktogramm

Array ausgeben
wiederhole für i = 0 bis len − 1
Ausgabe: himmelsrichtungen[i]
Nord
Ost
Süd
West

Die Schleifenvariable i durchläuft alle Index-Werte von 0 bis len(...) - 1 – egal, ob das Array 4 oder 10 000 Elemente hat, das Programm bleibt gleich lang.

Typische Muster: Größtes, Kleinstes, Durchschnitt

Mit demselben Schleifen-Prinzip lassen sich Arrays auswerten. Diese drei Muster brauchen Sie immer wieder (z. B. in den Übungen „Trainingsanalyse“ und „Wetterstation“): 

punkte = [5, 40, 3, 60, 20, 16]
laenge = len(punkte)

# Muster 1: Groesstes Element finden
groesster = punkte[0]
for i in range(laenge):
    if punkte[i] > groesster:
        groesster = punkte[i]

# Muster 2: Kleinstes Element finden
kleinster = punkte[0]
for i in range(laenge):
    if punkte[i] < kleinster:
        kleinster = punkte[i]

# Muster 3: Durchschnitt berechnen (Summe / Anzahl)
summe = 0
for i in range(laenge):
    summe = summe + punkte[i]
durchschnitt = summe / laenge

print("Bester Wurf:", groesster)
print("Schlechtester Wurf:", kleinster)
print("Durchschnittliche Punktzahl:", durchschnitt)
Bester Wurf: 60
Schlechtester Wurf: 3
Durchschnittliche Punktzahl: 24.0
📌 Merke: Das Suchmuster

Beim Suchen des größten (kleinsten) Elements startet man mit dem ersten Element als Zwischenstand und vergleicht dann der Reihe nach: Ist das aktuelle Element größer (kleiner) als der Zwischenstand, wird der Zwischenstand ersetzt.

4.7Tauschen, Einfügen, Entfernen

Zwei Elemente tauschen

Um zwei Werte zu vertauschen, braucht man einen Zwischenspeicher – sonst würde beim Überschreiben ein Wert verloren gehen (wie beim Umfüllen von zwei Gläsern: ohne drittes Glas geht es nicht): 

orte = ["Mannheim", "Stuttgart"]

zwischenspeicher = orte[0]
orte[0] = orte[1]
orte[1] = zwischenspeicher

print(orte)   # ['Stuttgart', 'Mannheim']

Ein Element an einer bestimmten Position einfügen

Python fügt neue Elemente mit append() grundsätzlich am Ende ein. Soll das neue Element an eine bestimmte Position, wird es anschließend Schritt für Schritt nach vorn getauscht, bis es an der richtigen Stelle steht. Beispiel: „Esslingen“ soll zwischen „Stuttgart“ und „Freiburg“ (also an Index 2):

orte = ["Mannheim", "Stuttgart", "Freiburg", "Konstanz"]

# zusaetzlicher Ort soll an Index 2
# Einfuegen an letzter Stelle
orte.append("Esslingen")

# Nach vorne wandern, bis an die richtige Position
for i in range(len(orte) - 1, 2, -1):
    zwischenspeicher = orte[i - 1]
    orte[i - 1] = orte[i]
    orte[i] = zwischenspeicher

print(orte)
['Mannheim', 'Stuttgart', 'Esslingen', 'Freiburg', 'Konstanz']

Die Schleife zählt rückwärts (Schrittweite −1) vom letzten Index bis zur Zielposition und tauscht das neue Element jeweils mit seinem Vorgänger.

Ein Element entfernen

Beim Entfernen ist es umgekehrt: Alle Elemente hinter der zu entfernenden Position rücken um eine Stelle nach vorn. Beispiel: „Stuttgart“ (Index 1) soll aus dem Array verschwinden: 

orte = ["Mannheim", "Stuttgart", "Esslingen", "Freiburg", "Konstanz"]

# zu loeschender Ort hat Index 1
for i in range(1, len(orte) - 1):
    orte[i] = orte[i + 1]

# letztes Element 'leeren'
orte[len(orte) - 1] = ""

print(orte)
['Mannheim', 'Esslingen', 'Freiburg', 'Konstanz', '']

Jedes Element überschreibt seinen Vorgänger. Das letzte Feld bleibt doppelt übrig und wird „geleert“. (Das zeigt übrigens gut, warum das Entfernen aus Arrays technisch aufwändiger ist als das Anhängen – in Kapitel 6 lernen Sie mit der verketteten Liste eine Struktur kennen, die genau das eleganter löst.)

Wurf: 40
Wurf: 3

Wurf: 20
Wurf: 16

Original: L1_3.3 Arbeitsauftrag Trainingsanalyse (BPE 7)

Lösung anzeigen 
wuerfe = []
for i in range(6):
    wuerfe.append(int(input("Wurf: ")))

bester = wuerfe[0]
schlechtester = wuerfe[0]
summe = 0

for i in range(len(wuerfe)):

        schlechtester = wuerfe[i]

print("Schlechtester Wurf:", schlechtester)

Übung 6 · Kapitalrechner mit Zinsstaffel schwer

Die Mühlberger Bank gewährt bei einer Anlage von mehr als 50.000 € eine Zinsstaffel: Jahr 1: 0,1 % · Jahr 2: 0,2 % · Jahr 3: 0,3 % · Jahr 4: 0,3 % · Jahr 5: 0,3 % · Jahr 6: 0,4 % · Jahr 7: 0,4 % · Jahr 8: 0,5 % · Jahr 9: 1 %.

Schreiben Sie ein Programm, das nach Eingabe von Laufzeit (1–9 Jahre) und Anlagebetrag den Auszahlungsbetrag (mit Zinseszins) berechnet. Beträge bis 50.000 € werden mit einer Meldung abgelehnt.

Tipp

Speichern Sie die Zinsstaffel in einem Array: zinsen = [0.1, 0.2, 0.3, 0.3, 0.3, 0.4, 0.4, 0.5, 1.0] – der Zinssatz von Jahr j steht dann an Index j - 1.

betrag = float(input("Geben Sie den gewünschten Anlagebetrag ein: "))

if betrag > 50000:
    for jahr in range(laufzeit):
        betrag = betrag * (1 + zinsen[jahr] / 100)
    print("Auszahlungsbetrag:", betrag, "EUR")
else:
    print("Der Anlagebetrag muss über 50.000 EUR sein!")

Übung 7 · Laufzeiten auswerten mittel

Beim Sporttag wurden die Zeiten für einen 400-m-Lauf in zwei Arrays gespeichert: namen enthält die Namen, zeiten die passenden Laufzeiten in Sekunden. 

namen = ["Mia", "Noah", "Lina", "Ben", "Sofia", "Emil"]
zeiten = [62.4, 58.1, 65.0, 57.8, 60.5, 59.9]

Schreiben Sie ein Programm, das ausgibt:

  1. die schnellste Person mit Zeit,
  2. die Durchschnittszeit,
  3. wie viele Personen unter 60 Sekunden geblieben sind.

Zusatzübung: Arrays gemeinsam auswerten (BPE 7)

Lösung anzeigen 
namen = ["Mia", "Noah", "Lina", "Ben", "Sofia", "Emil"]
zeiten = [62.4, 58.1, 65.0, 57.8, 60.5, 59.9]

bestzeit = zeiten[0]
best_index = 0
summe = 0
unter_60 = 0

for i in range(len(zeiten)):
    if zeiten[i] < bestzeit:
        bestzeit = zeiten[i]
        best_index = i
    if zeiten[i] < 60:
        unter_60 = unter_60 + 1
    summe = summe + zeiten[i]

durchschnitt = summe / len(zeiten)

print("Schnellste Person:", namen[best_index])
print("Bestzeit:", bestzeit)
print("Durchschnitt:", durchschnitt)
print("Unter 60 Sekunden:", unter_60)
Schnellste Person: Ben
Bestzeit: 57.8
Durchschnitt: 60.61666666666667
Unter 60 Sekunden: 3

Wichtig ist hier der Index: Wenn die beste Zeit an Index 3 steht, steht der passende Name ebenfalls an Index 3.
