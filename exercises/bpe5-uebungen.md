# Übungen zu BPE 5: Grundlagen der Programmierung

## Übung 1: EVA erkennen

Lies ein kleines Programm und markiere:

- Eingaben
- Verarbeitung
- Ausgaben

## Übung 2: Variablen verfolgen

Führe einen Schreibtischtest für ein Programm mit drei Variablen durch.

## Übung 3: Entscheidung formulieren

Formuliere zuerst eine Bedingung in Alltagssprache und dann als Python-Ausdruck.

## Übung 4: Schleife wählen

Entscheide, ob eine `for`-Schleife oder eine `while`-Schleife besser passt.
Begründe deine Wahl.

## Übung 5: Struktogramm zu Code
