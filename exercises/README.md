# Übungen

Dieser Ordner sammelt Aufgaben für das Lehrbuch.

Die Aufgaben sollen drei Stufen abdecken:

1. **Nachvollziehen:** Code lesen, Ablauf beschreiben, Schreibtischtest.
2. **Anwenden:** ähnliche Aufgabe selbst lösen.
3. **Übertragen:** neues kleines Problem strukturieren und begründen.

Aufgaben können später in die Kapitel übernommen oder als Übungsblätter
exportiert werden.
