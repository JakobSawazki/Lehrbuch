# Uebungen

Dieser Ordner sammelt Aufgaben fuer das Lehrbuch.

Die Aufgaben sollen drei Stufen abdecken:

1. **Nachvollziehen:** Code lesen, Ablauf beschreiben, Schreibtischtest.
2. **Anwenden:** aehnliche Aufgabe selbst loesen.
3. **Uebertragen:** neues kleines Problem strukturieren und begruenden.

Aufgaben koennen spaeter in die Kapitel uebernommen oder als Uebungsblaetter
exportiert werden.
