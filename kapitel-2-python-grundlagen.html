2.1Die Entwicklungsumgebung Thonny

Thonny ist eine kostenlose Entwicklungsumgebung für Python, die speziell für Einsteiger entwickelt wurde (Download: thonny.org – Python wird automatisch mitinstalliert). Der Bildschirm teilt sich in drei Bereiche:

BereichAufgabe
Editorfenster (oben) Hier geben Sie den Quellcode Ihres Programms ein. Nach dem Speichern starten Sie das Programm mit dem grünen Pfeil („Run current script“, Taste F5).
Ausgabe-/Eingabefenster („Shell“, unten) Hier erscheinen die Ausgaben Ihres Programms – und hier tippen Sie Eingaben ein, wenn das Programm danach fragt.
Hilfsfenster (rechts, über Menü „View“) Zusätzliche Ansichten, z. B. der Assistant (meldet Fehler und gibt Hinweise) oder der Variableninspektor.
💡 Tipp: Code-Vervollständigung nutzen

Drücken Sie beim Eintippen Strg + Leertaste – Thonny schlägt dann passende Befehle vor. So vermeiden Sie Tippfehler, eine der häufigsten Fehlerquellen!

2.2Ausgabe mit print()

🎯 Lernziel

Sie können Anweisungen zur Ausgabe von Zeichenketten entwickeln und als Struktogramm dokumentieren.

Fast jedes Programm zeigt dem Benutzer irgendetwas am Bildschirm an. In Python übernimmt das die Funktion print() – Ihr allererstes Python-Programm ist deshalb nur eine Zeile lang: 

print("Hallo Python-Programmierer!")
Hallo Python-Programmierer!

Struktogramm dazu

Begrüßung
Ausgabe: "Hallo Python-Programmierer!"

So funktioniert’s

print() gibt aus, was in den Klammern steht. Texte (Zeichenketten) müssen dabei in Anführungszeichen stehen – nur so erkennt Python, dass es sich um Text handelt und nicht um Programm-Elemente.

📌 Merke
  • Jede Anweisung steht in einer eigenen Zeile.
  • Zeichenketten stehen in doppelten oder einfachen Anführungszeichen: "Hallo" oder 'Hallo'.
  • Mit # beginnt ein Kommentar – er wird vom Programm ignoriert.

2.3Variablen – Schubladen für Daten

🎯 Lernziel

Sie können Variablen für Zeichenketten, Zahlen und Wahrheitswerte anlegen, ihnen Werte zuweisen und die Werte wieder auslesen.

Programme müssen sich Dinge merken: einen Namen, ein Alter, ein Zwischenergebnis. Dafür gibt es Variablen. Stellen Sie sich eine Variable wie eine Schublade vor: Sie hat einen Namen (das Schild an der Schublade) und einen Inhalt (den gespeicherten Wert). Über den Namen kommen Sie jederzeit wieder an den Inhalt heran.

Zeichenketten speichern

name = "Peter"   # Deklaration und Initialisierung
print(name)      # Ausgabe der Variablen auf dem Bildschirm
Peter
  • Zeile 1: Eine Variable namens name wird erzeugt (Deklaration) und erhält sofort den Wert "Peter" (Initialisierung). Das einzelne = ist das Zeichen für die Wertzuweisung.
  • Zeile 2: print(name) gibt den Inhalt der Variablen aus – ohne Anführungszeichen! print("name") würde dagegen wörtlich name ausgeben.
ℹ️ Datentyp? Macht Python selbst!

In Python muss der Datentyp bei der Deklaration nicht angegeben werden – er wird automatisch („implizit“) bei der Wertzuweisung festgelegt. "Peter" macht die Variable zur Zeichenkette (str), 3 zur Ganzzahl (int), 3.14 zur Dezimalzahl (float).

Regeln für Variablennamen

📌 Merke: Namensregeln
  • Am Anfang steht ein Kleinbuchstabe (keine Ziffer!).
  • Danach sind Buchstaben, Ziffern und Unterstriche erlaubt.
  • Nicht erlaubt: Umlaute (ä, ö, ü), ß, Leerzeichen, Sonderzeichen und Schlüsselwörter wie for.
  • Der Name soll etwas über den Inhalt aussagen. Zusammengesetzte Namen trennt man in Python mit Unterstrich: alter_in_jahren, gewicht_in_kg.

Mehrere Werte in einer Ausgabe

subjekt = "Ich"
praedikat = "spiele"
objekt = "Fußball"
print(subjekt, praedikat, objekt, "auf der Wiese")
Ich spiele Fußball auf der Wiese

In einer print()-Anweisung können mehrere Elemente stehen – Variablen und Zeichenketten gemischt, jeweils durch Kommas getrennt. Python fügt bei der Ausgabe automatisch ein Leerzeichen zwischen die Elemente ein.

Zahlen speichern

erste_zahl = 3       # Ganzzahl (int)
zweite_zahl = 3.14   # Dezimalzahl (float)
print(erste_zahl, zweite_zahl)
3 3.14
⚠️ Achtung: Punkt statt Komma

Dezimalzahlen werden in Python nach englischer Schreibweise mit einem Punkt geschrieben: 3.14, nicht 3,14.

Wahrheitswerte speichern

welt_ist_schoen = True
frieren_ist_schoen = False
print("Ist die Welt schön?", welt_ist_schoen)
print("Ist Frieren schön?", frieren_ist_schoen)
Ist die Welt schön? True
Ist Frieren schön? False

Neben Texten und Zahlen gibt es Wahrheitswerte (Typ boolean): True (wahr) und False (falsch).

⚠️ Achtung: Großschreibung!

Es muss genau True und False heißen – true oder false (klein geschrieben) führt zu einer Fehlermeldung.

2.5Eingabe mit input()

🎯 Lernziel

Sie können Benutzereingaben einlesen, in Zahlen umwandeln und in Berechnungen verwenden.

Bisher waren alle Werte fest im Programm eingebaut. Richtig nützlich werden Programme erst, wenn die Benutzer eigene Werte eingeben können. Dafür gibt es die Funktion input(): Sie zeigt eine Aufforderung an, wartet auf eine Tastatureingabe (abgeschlossen mit Enter) und liefert die Eingabe zurück, damit sie in einer Variablen gespeichert werden kann. 

vorname = input("Bitte Vornamen eingeben: ")
print("Willkommen", vorname)
Bitte Vornamen eingeben: Karin
Willkommen Karin

Struktogramm dazu

Begrüßung mit Eingabe
Eingabe: vorname
Ausgabe: "Willkommen", vorname

Das EVA-Prinzip

Dieses Muster heißt EVA-Prinzip und steckt in fast jedem Programm: Eingabe → Verarbeitung → Ausgabe.

Stolperfalle: input() liefert immer Text!

input() hat eine überraschende Eigenschaft: Alles, was eingegeben wird, behandelt Python als Zeichenkette – auch Ziffern! Was passiert, wenn man trotzdem damit rechnet? 

zahl = input("Bitte geben Sie eine Zahl ein: ")
haelfte = zahl / 2
print("Die Hälfte der eingegebenen Zahl ist:", haelfte)
Bitte geben Sie eine Zahl ein: 10
TypeError: unsupported operand type(s) for /: 'str' and 'int'

Die Fehlermeldung sagt: Die Division kann mit den Datentypen str (Zeichenkette) und int (Ganzzahl) nicht durchgeführt werden – eine Zeichenkette durch 2 zu teilen ergibt ja auch keinen Sinn!

📌 Merke: Datentypen umwandeln

Vor dem Rechnen muss die Eingabe in eine Zahl umgewandelt werden:

  • float(...) wandelt in eine Dezimalzahl um,
  • int(...) wandelt in eine Ganzzahl um.

Meist kombiniert man das direkt in einer Zeile: zahl = float(input("Zahl: "))

Das korrigierte Programm:

zahl = float(input("Bitte geben Sie eine Zahl ein: "))
haelfte = zahl / 2
print("Die Hälfte der eingegebenen Zahl ist:", haelfte)
Bitte geben Sie eine Zahl ein: 10
Die Hälfte der eingegebenen Zahl ist: 5.0

Geführtes Beispiel: Kreisfläche berechnen

So gehen Sie eine typische Aufgabe systematisch an – dieses Schema (Problemstellung → Problemanalyse → Struktogramm → Programmcode) begleitet Sie durch alle Aufgaben:

(I) Problemstellung: Ein Programm soll die Fläche eines Kreises berechnen. Der Benutzer gibt den Radius ein; das Programm gibt die Fläche aus. Formel: Kreisfläche = Radius · Radius · π

(II) Problemanalyse (Variablenliste):

BedeutungDatentypVariable
Radius des Kreises (Eingabe)floatradius
Kreiszahl π ≈ 3.14159floatpi
Kreisfläche (Ausgabe)floatflaeche

(III) Struktogramm:

2.6Funktionen – Programme sinnvoll gliedern

🎯 Lernziel

Sie können Funktionen definieren und aufrufen, Parameter übergeben und Rückgabewerte weiterverwenden.

Eine Funktion ist ein ausgelagertes Unterprogramm, das beliebig oft aufgerufen werden kann. Vorgänge, die immer wieder gebraucht werden, schreibt man so nur einmal – das macht Programme übersichtlicher und leichter zu ändern.

Stufe 1: Funktion ohne Parameter

# Hier ist die Funktion:
def hallo():
    print("Hallo miteinander!")

# Hier beginnt das Hauptprogramm:
hallo()
hallo()
hallo()
Hallo miteinander!
Hallo miteinander!
Hallo miteinander!
  • Die Definition beginnt mit dem Schlüsselwort def, danach folgen der Funktionsname, runde Klammern und ein Doppelpunkt.
  • Alle Anweisungen, die zur Funktion gehören, sind eingerückt.
  • Der Aufruf besteht aus Funktionsname und Klammern: hallo(). Bei jedem Aufruf springt das Programm in die Funktion, führt sie aus und kehrt danach zur Aufrufstelle zurück.
⚠️ Achtung

Die Funktion wird durch die Definition allein noch nicht ausgeführt! Ohne die Aufrufe im Hauptprogramm würde gar nichts ausgegeben.

Stufe 2: Funktion mit Parametern

Funktionen werden erst richtig mächtig, wenn man ihnen Informationen mitgeben kann – die sogenannten Parameter. Das folgende Programm begrüßt jeden Benutzer individuell mit Namen: 

def nutzer_gruessen(p_vorname, p_nachname):
    print("Herzlich willkommen,", p_vorname, p_nachname)

vorname = input("Bitte Vornamen eingeben: ")
nachname = input("Bitte Nachnamen eingeben: ")
nutzer_gruessen(vorname, nachname)
Bitte Vornamen eingeben: Karin
Bitte Nachnamen eingeben: Allgaier
Herzlich willkommen, Karin Allgaier
  • Die Funktion empfängt zwei Parameter (p_vorname, p_nachname), die in den Klammern stehen und durch Komma getrennt sind.
  • Beim Aufruf werden die konkreten Werte in den Klammern übergeben: nutzer_gruessen(vorname, nachname).
ℹ️ Warum das „p_“ vor dem Namen?

Zur besseren Unterscheidung von normalen Variablen bekommen Parameternamen in unseren Materialien ein vorangestelltes p_ (für „Parameter“). Das ist eine Konvention – Python selbst verlangt das nicht.

Stufe 3: Funktion mit Rückgabewert

Funktionen können nicht nur etwas ausgeben, sondern auch ein Ergebnis an die Aufrufstelle zurückliefern. Dafür gibt es das Schlüsselwort return. Beispiel: das Quadrat einer Zahl berechnen. 

def berechne_quadratzahl(p_zahl):
    quadratzahl = p_zahl * p_zahl
    return quadratzahl

zahl = float(input("Geben Sie bitte eine Zahl ein: "))
ergebnis = berechne_quadratzahl(zahl)
print("Die Quadratzahl von", zahl, "ist", ergebnis)
Geben Sie bitte eine Zahl ein: 7
Die Quadratzahl von 7.0 ist 49.0

return quadratzahl liefert den berechneten Wert an die Stelle zurück, von der die Funktion aufgerufen wurde – hier wird er in der Variablen ergebnis gespeichert und anschließend ausgegeben.

📌 Merke: Die drei Funktions-Stufen
  1. Ohne Parameter: tut bei jedem Aufruf genau dasselbe.
  2. Mit Parametern: bekommt Informationen übergeben und verarbeitet sie.
  3. Mit Rückgabewert: liefert mit return ein Ergebnis zurück, das weiterverwendet werden kann.

2.7Eine grafische Oberfläche (GUI)

🎯 Lernziel

Sie können den Programm-Code einer einfachen GUI erläutern und die Funktionalität eines GUI-Programms ergänzen.

Die meisten Programme, die Sie täglich benutzen, haben Fenster mit Eingabefeldern und Schaltflächen – eine grafische Benutzeroberfläche (englisch: Graphical User Interface, kurz GUI). Eine GUI besteht aus Komponenten:

KomponenteAufgabe
Fenster (Tk())Der Rahmen, in dem alle anderen Elemente sitzen.
LabelBeschriftungsfeld – zeigt Texte oder Ergebnisse an.
Entry (Textfeld)Eingabefeld, in das Benutzer Daten eintippen.
Button (Schaltfläche)Löst beim Anklicken eine Aktion aus – z. B. den Aufruf einer Funktion.

In Python liefert das Modul tkinter diese Komponenten. Hier das vollständige Beispielprogramm – ein Mini-Rechner, der eine eingegebene Zahl verdoppelt: 

from tkinter import *

# In dieser Funktion wird die eigentliche Berechnung durchgeführt.
# Aufruf der Funktion durch Klick auf die Schaltfläche
def berechnen():
    zahl_1 = float(tf_zahl1.get())
    ergebnis = zahl_1 * 2
    lb_ergebnis1_ausgabe.config(text=ergebnis)

# Hier beginnt das Hauptprogramm mit der Benutzeroberfläche
fenster = Tk()
fenster.title("Beispiel-GUI: Rechner")

# Elemente werden in einer Tabellenstruktur angeordnet:
# 1. Zeile
lb_zahl1 = Label(fenster, anchor=W, width=17, text="Zahl eingeben:")
lb_zahl1.grid(row = 0, column = 0)
tf_zahl1 = Entry(fenster)
tf_zahl1.grid(row = 0, column = 1)

# 2. Zeile
lb_ergebnis1 = Label(fenster, text="Verdopplung der Zahl:")
lb_ergebnis1.grid(row = 1, column = 0)
lb_ergebnis1_ausgabe = Label(fenster, bg="white", relief = SUNKEN,
                             anchor=W, width = 17)
lb_ergebnis1_ausgabe.grid(row = 1, column = 1)

# Schaltfläche in der 3. Zeile rechts
bt_umrechnen = Button(fenster, text="Berechnung starten",
                      command=berechnen)
bt_umrechnen.grid(row = 2, column = 1)

So ist das Programm aufgebaut

Das Programm hat zwei klar getrennte Teile:

Teil 1: Die Oberfläche bauen

  • from tkinter import * bindet das GUI-Modul ein.
  • fenster = Tk() erzeugt das Fenster, fenster.title(...) beschriftet die Titelzeile.
  • Jedes Element wird erst erzeugt (z. B. Label(fenster, text="...")) und dann mit .grid(row=…, column=…) in einer Tabellenstruktur platziert.
  • Beim Button legt command=berechnen fest, welche Funktion beim Klick aufgerufen wird (Achtung: ohne Klammern!).
⚠️ Achtung: Zählung beginnt bei 0

Bei der Tabellenanordnung mit grid() beginnt Python bei 0 zu zählen: Die erste Zeile ist row = 0, die erste Spalte column = 0.

Teil 2: Die Funktionalität

Die Funktion berechnen() wird bei jedem Klick auf die Schaltfläche ausgeführt:

  1. tf_zahl1.get() liest den Inhalt des Eingabefelds aus; float(...) macht daraus eine Dezimalzahl.
  2. Die Zahl wird verdoppelt und in ergebnis gespeichert.
  3. lb_ergebnis1_ausgabe.config(text=ergebnis) zeigt das Ergebnis im Ausgabe-Label an.
ℹ️ Auch hier: EVA-Prinzip

Eingabe (Textfeld auslesen) → Verarbeitung (verdoppeln) → Ausgabe (Label beschriften). Nur die Werkzeuge sind andere als bei der Konsole: tf.get() statt input(), lb.config(text=…) statt print().

Übung 6 · Urlaubsrechner mit Funktion schwer

Sie möchten mit dem Auto in den Urlaub fahren. Schreiben Sie ein Programm, das die Benzinkosten der Fahrt berechnet. Die Berechnung soll in eine Funktion mit Parametern und Rückgabewert ausgelagert werden. Die Funktion erhält: Benzinpreis, Verbrauch des Autos (l/100 km) und Länge der Strecke (km).

Tipp zur Verarbeitung

gesamtverbrauch = strecke * verbrauch_auf_100km / 100
kosten = gesamtverbrauch * benzinpreis

Original: L2_5_1 Aufgabe Funktion Urlaubsfahrt

Lösung anzeigen 
def berechne_benzinkosten(p_benzinpreis, p_verbrauch, p_strecke):
    gesamtverbrauch = p_strecke * p_verbrauch / 100
    kosten = gesamtverbrauch * p_benzinpreis
    return kosten

benzinpreis = float(input("Benzinpreis pro Liter (Euro): "))
verbrauch = float(input("Verbrauch auf 100 km (Liter): "))
strecke = float(input("Länge der Strecke (km): "))

benzinkosten = berechne_benzinkosten(benzinpreis, verbrauch, strecke)
print("Die Benzinkosten betragen:", round(benzinkosten, 2), "Euro")
Benzinpreis pro Liter (Euro): 1.85
Verbrauch auf 100 km (Liter): 6.5
Länge der Strecke (km): 750
Die Benzinkosten betragen: 90.19 Euro

Das Hauptprogramm liest die drei Werte ein, übergibt sie der Funktion und gibt den zurückgelieferten Wert gerundet aus.

Übung 7 · Höhenmesser als Funktion schwer

Bauen Sie Ihr Höhenmesser-Programm aus Übung 5 um: Die Berechnung soll jetzt in einer Funktion berechne_hoehe() stattfinden. Sie empfängt die Siedetemperatur als Parameter, berechnet die Höhe und gibt sie als Rückgabewert zurück.

Original: L2_5_2 Übungsaufgabe Höhenmeter Funktion

Lösung anzeigen 
def berechne_hoehe(p_siedetemperatur):
    hoehe = 30000 - 300 * p_siedetemperatur
    return hoehe

print("Programm zur Berechnung der Höhe anhand der Siedetemperatur")
siedetemperatur = float(input("Siedetemperatur in Grad Celsius: "))
hoehe = berechne_hoehe(siedetemperatur)
print("Die Höhe beträgt:", hoehe, "Meter")

Die Ausgabe ist dieselbe wie in Übung 5 – aber die Berechnung ist jetzt sauber ausgelagert und könnte von überall im Programm wiederverwendet werden.

Übung 8 · Temperatur-Rechner mit GUI schwer

In den USA werden Temperaturen in Fahrenheit angegeben. Vervollständigen Sie den GUI-Temperatur-Rechner: Die Oberfläche (Eingabefeld tf_fahrenheit, Ausgabe-Label lb_celsius_ausgabe, Schaltfläche) ist schon fertig – es fehlt nur die Funktion umrechnen().

Recherchieren Sie die Umrechnungsformel von Fahrenheit nach Celsius und ergänzen Sie die Funktion.

Tipp zur Formel

celsius = (fahrenheit − 32) · 5 / 9

Original: L2_6 Aufgabe Anwendungsbeispiel GUI

Lösung anzeigen 
def umrechnen():
    fahrenheit = float(tf_fahrenheit.get())
    celsius = (fahrenheit - 32) * 5 / 9
    lb_celsius_ausgabe.config(text=round(celsius, 2))

Ablauf wie immer nach dem EVA-Prinzip: Eingabefeld auslesen und umwandeln → Formel anwenden → Ergebnis (gerundet) im Ausgabe-Label anzeigen.
